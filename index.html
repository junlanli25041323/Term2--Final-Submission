Default Human: Bias in Image Generation Systems

This project aims to explore how image generation models can generate default human images due to biases in the data, classification, and output. Furthermore, the project tests whether these biases can be made visible and adjustable through intervention.

01The Default Human

A Professional person

A Successful person

A CEO

The terms "professional person," "successful person," and "CEO" are different types of descriptions. However, the results output by the AI image generator are highly similar. The images all depict a young, well-dressed, smiling man. This suggests that the model has a default image.

02Not All Prompts Behave the Same

Professional person

Leader

Teenager

When I entered the word "teenager," the generated images clearly reflected differences in age and appearance. This indicates that the model can distinguish between different categories. However, abstract character descriptions often exhibit similar representations.

03The category is assumed

Categories such as "professional," "successful person," and "CEO" are not entirely neutral concepts. These terms are assumed to represent a specific image. These assumptions influence the features of AI-generated images.

04Repetition and Limited Variation

Repeated output

After repeatedly inputting the same prompts, the output consistently tends to resemble similar faces and styles. Despite differences in scenes or clothing, the overall features remain similar. This indicates that the model has limited surface features.

05What Is Missing and What Is Controlled

Although the output changes, images possessing certain identities and appearances consistently fail to appear. This demonstrates that while the system actively restricts certain types of content, it fails to address this bias. This makes some inequalities obvious, while others remain hidden.

06Making the Bias Adjustable

After adjustment: diverse genders, body types

Introducing diversity-related terms shifts the output, but not in a straightforward way. Instead of generating diverse individuals, the model often represents diversity through groups of people. This suggests that diversity is encoded as a visual pattern rather than evenly distributed across individual representations.

07Theory of Change

This project demonstrates that biases in image generation affect the entire computational process, from data and classification to model output. By continuously inputting new instructions, users can constantly adjust the output they obtain.
